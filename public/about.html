About CEO AI

CEO AI is a satirical demonstration that explores how AI might replace executive decision-making. Toggle between three modes to see how the same question gets answered in Public (corporate buzzwords), CEO (direct business-speak), or Private (brutally honest) contexts.

The Tech Stack (Or: What Makes This Tick)

Let's be honest—this is just a fancy wrapper around an AI API. Here's everything that powers this satirical experiment:

  • Google Gemini 2.5 Flash - The actual AI doing all the work
  • Node.js + Express - Basic backend server
  • Vercel - Serverless hosting platform
  • GitHub - Code repository
  • Namecheap - Domain registration
  • Buy Me a Coffee - Monetization (to cover API costs)
  • HTML/CSS/JavaScript - The classic trio

Built as a tongue-in-cheek exploration of AI's role in business leadership. No actual CEOs were replaced in the making of this website.
