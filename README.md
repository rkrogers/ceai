# CEAI - CEO AI

A satirical web application that uses AI to respond like a CEO in different contexts.

## About

CEO AI is a humorous take on corporate communication, offering three distinct modes:
- **Public Mode**: Corporate PR-friendly responses full of buzzwords
- **CEO Mode**: Direct, bottom-line focused executive speak
- **Private Mode**: Brutally honest, unfiltered responses

## Prerequisites

- Node.js 14+ and npm
- Google Gemini API key (free)

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/rkrogers/ceai.git
   cd ceai
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Set up your Gemini API key:**
   
   a. Get a free API key from Google: https://makersuite.google.com/app/apikey
   
   b. Create a `.env` file in the project root:
   ```bash
   cp .env.example .env
   ```
   
   c. Edit `.env` and add your API key:
   ```
   GEMINI_API_KEY=your_actual_api_key_here
   PORT=3000
   ```

4. **Start the server:**
   ```bash
   npm start
   ```

5. **Open your browser:**
   ```
   http://localhost:3000
   ```

## Usage

1. Select a mode (Public, CEO, or Private)
2. Type your question in the search box
3. Click "Ask CEO" to get a response
4. Watch the AI respond in the selected mode's style

## Example Questions

Try asking:
- "Should we increase employee salaries?"
- "What do you think about work-life balance?"
- "How do we handle this PR crisis?"
- "Why are profits down this quarter?"

Each mode will respond very differently!
